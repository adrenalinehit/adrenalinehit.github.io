# What's this all about then?

This is my new dev blog - I've had a few false starts, and have considered writing on one of the other platforms. Setting it up from scratch myself seemed like a fun alternative.

I want to try and chronicle some of my later journeys through software development, focussing on the technical side of things (although please excuse me if I stray off topic sometimes!)

I'm playing with VSCode (amazing to see how quickly it's developed into a serious contender), the Ruby/Rails ecosystem and github so github pages and Jekyll seemed like the obvious choice.  Seeing the live preview to my right is pretty cool as I type this too.  The "no server" approach is nice and easy too..

They'll be the usual investigatory type posts (aide memoires for me and the internet - to be sure, I've used a few so far!) and I'm going to carry on looking into some of the newer languages too (coz everyone needs a break from dotnet every now and then).
